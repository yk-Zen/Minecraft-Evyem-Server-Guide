GUIDE FOR SETTING UP MINECRAFT

  1. Go to GDrive: Link to download mods in one click.

  2. Go to SKLaucher Website: SKLauncher and Install SKLauncher on your computer.

  3. Open SKLauncher and click 'Switch to offline mode'.

  4. Enter your username and click log in.

  5. After logging in, click 'Installations Manager'

  6. Click 'Add' to create a new installation.

  7. Name your installation 'Fabric 1.21.11', select the Fabric version and last put a check mark

  8. Click 'Save' to save your installation.

  9. Go back, select the Fabric 1.21.11 and click 'Play' to launch Minecraft.

  10. After lauching, quit the game.

  11. Go back to the installations manager, hover the Fabric 1.21.11 and click the folder icon.

  12. Open the 'mods' folder and move/paste all the downloaded mods into it.

  13. Go back to SKLauncher and click 'Play' to launch Fabric 1.21.11 with the mods.

  14. Select Multiplayer, add Server and enter the server IP address. Click Done.

  15. Finally, click Join Server YEYYY.
